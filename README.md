# Random Task Generator

**Автор:**  Климова Наталья

## Описание

Random Task Generator — это десктопное приложение на Python с графическим интерфейсом (tkinter), которое помогает случайным образом выбирать задачу из списка. Приложение сохраняет историю всех сгенерированных задач в JSON-файл, поддерживает фильтрацию по типу задачи (учёба, спорт, работа) и позволяет пользователю добавлять собственные задачи с проверкой на пустую строку.

## Возможности

- Генерация случайной задачи из предопределённого списка.
- Отображение истории с временными метками.
- Фильтрация истории по типу задачи.
- Добавление новых задач пользователем (проверка на непустую строку).
- Сохранение и загрузка истории в/из JSON.
- Очистка истории с подтверждением.

## Требования

- Python 3.6+
- Стандартные библиотеки: `tkinter`, `random`, `json`, `os`, `datetime`
